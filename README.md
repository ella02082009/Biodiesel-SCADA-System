# Biodiesel-SCADA-Dashboard-
## 🚀 Quick Start / Local Development

To run the SCADA Dashboard prototype locally, make sure your development server is running in your terminal, then click the link below:

👉 [Launch Local SCADA Dashboard](http://localhost:5173) *(or change to http://localhost:3000 depending on your port)*

### How to spin it up:
1. Open your terminal in the project directory.
2. Run `npm install` to ensure dependencies are ready.
3. Fire up the development server:
   ```bash
   npm run dev
